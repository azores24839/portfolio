# Vercel + Cloudflare R2 部署设计

## 目标

将当前约 1.5GB 的静态作品集部署上线，尽量保持零费用，并让图片、影片、PDF、字体都能正常访问。教程面向没有部署经验的使用者，登录、付款方式验证和密钥复制等必须人工完成的步骤需要明确标注。

## 已确认的项目现状

- 网站是由 `index.html`、`styles.css`、`app.js` 和自动生成的 `portfolio-data.js` 组成的纯静态网站。
- 项目约 1.52GB，其中影片约 698MB、图片约 602MB，其他文件约 224MB。
- Vercel Hobby 的 CLI 单次源文件上传上限为 100MB，不能直接上传整个目录。
- 项目当前不是 Git 仓库，因此还没有连接 GitHub 或 Vercel 的自动部署流程。

## 选定架构

### Vercel

Vercel 只接收运行网站必需的小文件：HTML、CSS、JavaScript，以及部署配置。部署包不包含本地媒体、PDF和字体原文件，因此保持在免费方案限制内。

### Cloudflare R2

R2 保存公开媒体文件，包括：

- 项目图片与影片
- 校园照片
- 简历图片与 PDF
- 页面装饰图片和纹理
- 自托管字体

R2 使用 Standard 存储类型。当前约 1.5GB，低于每月 10GB-month 的免费储存额度；R2 直接对外传输不收流量费。用户需要在 Cloudflare 控制台开通 R2，并完成付款方式验证。

## URL 与构建方式

增加一个公开的媒体基础地址，例如：

`https://media.example.com/`

本地开发时，媒体继续读取仓库里的相对路径。执行部署构建时，生成脚本把媒体路径转换为：

`R2 基础地址 + 原相对路径`

这样可以保留当前目录组织，不需要逐张手改图片地址。Vercel 通过环境变量保存 R2 公共基础地址，避免把账号密钥写进源码。公开媒体 URL 不属于秘密，但上传所用的 Access Key 和 Secret Key 必须只保存在本机或安全的 CI 环境中。

## 部署流程

1. 用户注册或登录 Cloudflare，开通 R2。
2. 创建公开 Bucket，首次可使用 Cloudflare 提供的 `r2.dev` 地址；需要正式域名时再绑定自定义媒体子域名。
3. 使用上传脚本将项目媒体同步到 R2，并保持原目录层级。
4. 在本地验证抽样图片、影片、PDF和字体 URL。
5. 初始化 Git 仓库，并通过 GitHub Desktop 或命令行推送到 GitHub；媒体目录由 `.gitignore` 排除，避免超过 GitHub 的 100MB 单文件限制。
6. 用户登录 Vercel，通过 GitHub 导入仓库。
7. 在 Vercel 设置 R2 公共基础地址并部署。
8. 验证首页、所有作品分类、影片播放、PDF、字体、移动端和刷新行为。

## 更新作品的日常流程

用户仍按当前方式把新作品放入 `content/projects/.../media`，然后运行一个统一的部署准备命令。该命令会：

1. 校验作品内容；
2. 上传新增或变更的媒体到 R2；
3. 重新生成 `portfolio-data.js`；
4. 检查 Vercel 部署包不包含大型媒体。

之后只需提交并推送小型代码文件，Vercel 会自动重新部署。

## 错误处理与安全

- 上传脚本缺少 R2 配置时立即停止，并提示缺少哪个变量。
- R2 上传失败时不删除本地文件，也不继续发布不完整的数据。
- `.env`、R2 Access Key 与 Secret Key 必须被 Git 忽略。
- Bucket 只允许公开读取，不允许匿名写入。
- 上传凭据采用最小权限，只授权指定 Bucket。
- 为媒体设置正确的 `Content-Type` 和浏览器缓存头，确保影片可播放、字体可加载。

## 验证标准

- `npm run check` 通过。
- 生产构建目录明显小于 Vercel Hobby 的 100MB 限制。
- 抽样检查 JPG、PNG、MP4、PDF、WOFF/WOFF2 均返回正确状态与类型。
- 本地开发不依赖联网也能查看本地媒体。
- Vercel 正式网址中所有作品内容可见，影片可播放。
- Git 状态中不出现密钥和大型媒体文件。

## 费用边界

- Vercel Hobby：个人作品集免费使用，超过免费用量时会停止相关资源，而不是自动购买额外 Hobby 用量。
- Cloudflare R2 Standard：每月包含 10GB-month 储存、100万次 Class A、1000万次 Class B，直接外网流量免费。
- 当前媒体规模与普通作品集访问量预计落在免费额度内；仍建议在 Cloudflare 设置用量通知并定期查看账单。
